Программа обучения
Блок 1.
Язык гипертекстовой разметки. Структура HTML-документа. HTML-формы. CSS3. Селекторы, атрибуты, псевдоклассы. Блочная вёрстка.
Блок 2.
Синтаксис. Управляющие конструкции и операторы. Пользовательские функции. Область видимости. Переменные и массивы. Классы и объекты. Сессии. Работа с файловой системой. Динамическая генерация страниц на стороне сервера.
Блок 3.
Синтаксис. Управляющие конструкции и операторы. Функции и понятие контекста функции. Область видимости. Концепция объектной модели (DOM). Понятие прототипа и прототипного наследования. Манипуляции над объектами дерева DOM. События и обработка событий. Библиотека jQuery. Библиотека Lodash. Ajax.
Блок 4.
Разработка в CMS Битрикс. Модули. Компоненты. API D7. Работа с сущностями (getlist / add / update / delete)
Как происходит обучение
Вы посещаете лекции и практические занятия 2-3 раза в неделю
Преподаватели подбирают материалы для самостоятельного изучения
Эксперты курса проверяют ваши домашние задания и Вы получаете обратную связь
